# ДЗ_02.Телефонная книга (MVC - рефакторинг ДЗ_01)

Простой консольный проект телефонной книги, реализованный (попытка реализовать) по паттерну MVC.

## Структура проекта

- `main.py` — точка входа
- `hw_model.py` — модели (данные)
- `hw_view.py` — отображение (меню, вывод)
- `hw_controller.py` — логика работы
- `hw_utils.py` — вспомогательные функции ввода
- `phone_book.csv` — файл с контактами

## Как запустить

1. Перейди в папку проекта
2. Выполни команду:

```bash
python3 main.py
